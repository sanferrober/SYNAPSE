### Si persisten problemas:
1. **Verificar permisos**: Ejecutar como administrador
2. **Actualizar pip**: `python -m pip install --upgrade pip`
3. **Limpiar cache**: `pip cache purge`
4. **Usar entorno virtual**: `python -m venv synapse_env`
